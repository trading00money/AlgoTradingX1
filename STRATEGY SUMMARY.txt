  SECTION 1 — STRATEGY OVERVIEW
================================================================

NAME      : v5 EMA20 Pullback Strategy
EDGE      : Enter on pullbacks to dynamic support (EMA20) within
            confirmed macro trends, confirmed by Ehlers DSP state
TIMEFRAME : H1 (1-hour bars)
INSTRUMENT: XAUUSD / GOLD CFD (MT5 symbol: GOLD)
DATA FEED : 3,500 H1 bars required for indicator convergence
            (bottleneck: D1 EMA50 needs 115 daily bars = 2,760 H1)

ARCHITECTURE PHILOSOPHY
  Unlike v1–v4 which entered on indicator crossovers (timing
  problem), v5 enters on PULLBACKS TO DYNAMIC SUPPORT within
  confirmed trends. This gives:
    - Better entry price (at or near support)
    - Tighter stop loss placement (support is close to entry)
    - Higher statistical reliability (trend continuation from
      support > crossover timing)
    - Ehlers STATE filters (not crossovers) confirm direction
      without timing noise

================================================================
  SECTION 3 — ENTRY SIGNAL LOGIC
================================================================

  File: core/signal_engine.py → generate_signals_for_backtest()

  LONG ENTRY (signal = 1):
    All of the following must be true on the same H1 bar close:

    Pullback trigger:
      low  <= EMA20 + 0.5 × ATR(14)   (price touched EMA20 zone)
      low  >= EMA20 - 1.0 × ATR(14)   (not too far below EMA20)
      close > EMA20                    (bar closed above EMA20 → support held)

    Ehlers state confirmation (all 3):
      MAMA > FAMA                      (adaptive trend: bullish)
      ITrend > ITrend_trigger          (DSP trendline: bullish)
      EBSW > -0.5                      (Even Better Sinewave: not in downturn)

    Regime gate (all 5):
      close > EMA200 AND EMA200_slope > 0   (H1 macro trend: bullish)
      D1 close > D1 EMA50                   (intermediate trend: bullish)
      DI+ > DI- + 1                         (directional: +1 buffer)
      ADX(14) > 15                          (moderate trend required)
      07:00 <= bar_hour < 20:00 UTC         (London + NY session)

  SHORT ENTRY (signal = -1):
    Mirror of long, with conditions inverted:
      high >= EMA20 - 0.5 × ATR(14)
      high <= EMA20 + 1.0 × ATR(14)
      close < EMA20                    (bar closed below EMA20 → resistance held)
      MAMA < FAMA, ITrend < trigger, EBSW < 0.5
      close < EMA200, D1 below D1 EMA50, DI- > DI+ + 1, ADX > 15, session OK

  COOLDOWN: 5-bar global cooldown between any two signals (same
  direction or opposite). Prevents repeated entries in choppy zones.


================================================================
  SECTION 4 — INDICATOR SPECIFICATIONS
================================================================

  EMA20    : close.ewm(span=20, adjust=False)  — pullback target
  EMA200   : close.ewm(span=200, adjust=False) — H1 macro regime
  EMA200_slope: ema200 - ema200.shift(20)      — momentum direction
  D1 EMA50 : daily close resampled to D, ewm(span=50), ffilled to H1
  ATR(14)  : EMA-smoothed true range, alpha=1/14 (Wilder)
  ADX(14)  : Wilder-smoothed DX, alpha=1/14
  DI+/DI-  : EMA-smoothed directional movement, alpha=1/14

  MAMA     : Mesa Adaptive Moving Average (modules/ehlers/mama.py)
             Adaptive phase-locked trend following
  ITrend   : Instantaneous Trendline, alpha=0.07
             (modules/ehlers/instantaneous_trendline.py)
  EBSW     : Even Better Sinewave (modules/ehlers/sinewave_indicator.py)
             HP(40) + SuperSmoother(10) cycle detector

  Session filter : hour >= 7 AND hour < 20 (UTC)
                   Includes London (07:00–16:00) and NY (13:00–20:00)


================================================================
  SECTION 5 — EXIT LOGIC
================================================================

  MT5 SL/TP (hard levels, set at entry):
    SL = entry ± ATR(14) × 3.0 (effective multiplier, see vol-adaptive below)
    TP = entry ± SL_distance × 5.0

  Partial exit at 3R:
    When unrealized profit >= 3.0 × initial_risk:
      Close 50% of position at market
      Move SL to breakeven + 0.3R buffer
    Rationale: at 5R TP, 2R is too early; 3R = 60% of the way to TP,
               sufficient confirmation trade is working

  Time stop at 30 bars:
    If bars_open >= 30 AND unrealized_r < 0:
      Close 100% of position at market
    Rationale: converts large SL losses into smaller time stop losses;
               sideways losers exit early, reducing avg_loss

  Fast regime exit (Ehlers-based):
    Exit uses majority-vote Ehlers: any 2 of 3 Ehlers bearish → exit long
    Exit long  if ehlers_bear_count >= 2
    Exit short if ehlers_bull_count >= 2
    Rationale: catches reversals 5–15 bars earlier than slow EMA200/ADX;
               fast regime is entry-gate only for the opposite direction

  Note: SL/TP are set in MT5 at entry. Regime exit and time stop
  are applied programmatically by the script each bar close.
  Trailing stop: DISABLED (tested; hurts PF at all settings)
  Breakeven stop: DISABLED (cuts avg_win more than avg_loss; PF 1.90→1.83)


================================================================
  SECTION 6 — POSITION SIZING
================================================================

  Method     : Fixed fractional (ATR-derived, not fixed pips)
  Formula    : lots = (equity × risk_pct / 100) / (SL_distance × 100)
               (GOLD contract = 100 troy oz/lot on MT5)

  Live paper : 0.5% equity risk per trade  (backtest used 1.0%)
  Minimum    : 0.01 lots
  Maximum    : 50.0 lots

  Short sizing: 0.5× normal (50% of long risk per short trade)
    Rationale: shorts add diversification/alpha but have higher
               variance; asymmetric sizing limits drag on equity curve

  Seasonal sizing (April–September):
    25% of normal size (0.5% × 0.25 = 0.125% effective risk)
    Rationale: walk-forward shows all 4 failing folds are Q2/Q3;
               range-bound gold in these months compresses realised RR


================================================================
  SECTION 7 — ENTRY FILTERS (RUNTIME GUARDS)
================================================================

  1. Vol-adaptive SL multiplier
     ATR percentile rank over rolling 200-bar window:
     If ATR_pct > 70th percentile:
       Scale multiplier from 3.0 down toward 1.5 (floor)
     Prevents outsized dollar-losses during abnormally volatile regimes

  2. MA(200) trend direction filter (year-round)
     Block long  if close < MA200
     Block short if close > MA200
     Rationale: ADX can be high in a down-trending market (e.g. 2022
                gold crash). This filter prevents fighting the macro trend.

  3. Seasonal ADX gate (April–September only)
     Block entry entirely if ADX(14) < 20
     Rationale: ADX < 20 = no directional trend in weak season →
                signals fire into chop → deeply negative Sharpe

  4. SL hard cap: $100/oz maximum SL distance
     Prevents absurdly wide stops at extreme volatility spikes

  5. Session filter: 07:00–19:59 UTC only
     Outside London+NY: liquidity thin, spreads wide, signals unreliable
